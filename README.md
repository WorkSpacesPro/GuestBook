创建项目根目录（例如 guestbook/）。

在根目录下创建 index.html，并粘贴上面的 HTML 代码。

创建 assets/css/ 文件夹，放入 style.css。

创建 assets/js/ 文件夹，放入 main.js。

创建 assets/img/ 文件夹（可留空）。

使用任意 Web 服务器（如 npx serve . 或 VS Code Live Server）运行 index.html。
